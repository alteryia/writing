# rainy

_a haiku collection about rain_

## puddles

```
rainy day dance-off
puddle jumping, soggy shoes
splashy victory!
```

## raindrops

```
raindrops on the roof
tiny drummers out of sync
a scattered rhythm
```

## rainy

```
rain's sudden downpour
forgot my umbrella - oops
now i'm a wet mop
```
